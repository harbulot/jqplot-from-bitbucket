These examples show rectangle overlays to show various ranges. xmin, xmax, ymin, ymax are all optional and the rectangle will go to the end of the plot area if they are not specified. Colours are specified in the canvas drawing format (use rgba for transparency). 





Ideally, this would word directly with values like "2012-01-03", without an explicit conversion to a Date object. (It works with milliseconds since Epoch.)
